Big-O on canvas

This page shows how fast various algorithms are with different input sizes. While sorting 5 books is fast, sorting 500 books is much, much slower (not 100, but 10000 times slower).

But, if you have some tricky sorting method (blue line), it will be faster than comparing every book to every other (green line).
